# Política de Privacidade - COMPIA Enterprise

> **Versão:** 1.0 (DRAFT - LGPD Compliant)
> **Última Atualização:** 04/02/2026
> **Status:** Minuta para Revisão Jurídica

---

## 1. Introdução

O **COMPIA Enterprise** ("nós", "nosso") está comprometido em proteger a privacidade e os dados pessoais de seus usuários, em conformidade com a **Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018)**.

Esta política descreve como coletamos, usamos, armazenamos e compartilhamos seus dados.

## 2. Dados Coletados

### 2.1 Dados de Cadastro
*   Nome completo;
*   Endereço de e-mail corporativo;
*   Cargo/Função;
*   Telefone (opcional).

### 2.2 Dados de Uso
*   Logs de acesso (IP, Data, Hora) - Obrigatório pelo Marco Civil da Internet;
*   Dados de geolocalização (apenas durante o uso do recurso de inspeção georreferenciada);
*   Metadados de arquivos (fotos/vídeos) enviados.

## 3. Finalidade do Tratamento

Utilizamos seus dados para:
1.  **Prestação do Serviço:** Autenticar usuários, processar inspeções e gerar relatórios.
2.  **Segurança:** Auditar acessos suspeitos e prevenir fraudes.
3.  **Melhoria do Produto:** Analisar métricas de uso anonimizadas para otimizar a performance.
4.  **Comunicação:** Enviar notificações sobre o sistema, atualizações de segurança ou respostas a chamados de suporte.

## 4. Compartilhamento de Dados

Não vendemos seus dados. Compartilhamos informações apenas com subprocessadores essenciais para a operação:

| Subprocessador | Finalidade | Localização |
| :--- | :--- | :--- |
| **Supabase Inc.** | Banco de Dados e Autenticação | EUA / AWS |
| **Vercel Inc.** | Hospedagem do Frontend | Global (CDN) |
| **OpenAI / Google** | Processamento de Inteligência Artificial | EUA |

Todos os subprocessadores possuem contratos de confidencialidade e segurança adequados.

## 5. Seus Direitos (Titular dos Dados)

Você tem direito a:
*   Confirmar a existência de tratamento de dados;
*   Acessar seus dados;
*   Corrigir dados incompletos ou desatualizados;
*   Solicitar a exclusão de dados desnecessários;
*   Revogar o consentimento (quando aplicável).

Para exercer seus direitos, entre em contato com nosso Encarregado de Dados (DPO) através do e-mail: `dpo@compia.tech`.

## 6. Segurança da Informação

Adotamos medidas técnicas robustas, incluindo:
*   Criptografia de dados em trânsito (TLS);
*   Criptografia de dados em repouso (Banco de Dados);
*   Controles de acesso rígidos (RLS - Row Level Security);
*   Backups frequentes.

## 7. Retenção de Dados

Manteremos seus dados enquanto você tiver uma conta ativa ou pelo tempo necessário para cumprir obrigações legais (ex: guarda de logs por 6 meses/5 anos conforme legislação).
